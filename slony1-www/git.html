Git Repository Access

The slony1-engine CVS repository is now obsolete, as future development will take place using Git.

The official Git repository is found at git.postgresql.org; specifically see slony1-engine.git Summary for more details.

There are several addresses at which one may get at the repository via the various protocols that Git supports:

  • git://git.postgresql.org/git/slony1-engine.git
  • http://git.postgresql.org/git/slony1-engine.git
  • ssh://git@git.postgresql.org/slony1-engine.git

There is also likely to be a clone at GitHub.com.
